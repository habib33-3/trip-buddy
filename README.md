## ğŸš€ Quick Start

### Prerequisites

- Node.js (v18+)
- PostgreSQL (v15+)
- PNPM (v8+)

### Setup

1. **Clone & Install**

   ```bash
   git clone https://github.com/habib33-3/trip-buddy.git
   cd trip-buddy
   pnpm install
   ```

2. **Configure Environment**  
   Duplicate `.env.example` to `.env` and add your credentials.

3. **Run the App**

   ```bash
   pnpm build
   pnpm start  # Starts dev server
   ```

4. **Open in Browser**  
   Visit `http://localhost:4173`
